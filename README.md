# Reframe / - Webエンジニア ポートフォリオ

ビジネスの枠組みを再定義（Reframe）し、設計から実装まで一気通貫でサポートする並木健太のWeb名刺・ポートフォリオサイトです。

## 🚀 Concept
**「無理のない前進」をつくるための再定義。**
リラクゼーションの世界で学んだ「視点を変えれば景色が変わる」という経験を活かし、技術的な課題だけでなく、ビジネスの本質的な枠組みを組み替える実装パートナーを目指しています。

## 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS
- **Animation:** Framer Motion
- **Deployment:** Vercel

## 📂 Project Highlights
- **会議アクショントラッカー:** 会議の決定事項を確実にアクションへ繋げるタスク管理ツール。
- **LINE × Notion 業務連携:** 現場入力をLINEで完結させ、Notionへ自動集約するシステム。

## 📱 Features
- **Responsive Design:** PCからスマホまで、あらゆるデバイスで最適化された閲覧体験。
- **Interactive UI:** Framer Motionによる滑らかなオープニングアニメーションとホバー演出。
- **Static Typing:** TypeScriptによる堅牢なコード設計。
